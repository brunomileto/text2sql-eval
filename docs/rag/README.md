# Corpus RAG

Este diretório armazena documentos-fonte curados para o subsistema reutilizável
de RAG.

O estilo desejado não é o de uma exportação técnica de metadados. Esses arquivos
devem parecer páginas internas de wiki voltadas para analistas, como as que uma
organização real manteria sobre seus dados de negócio e práticas de reporting.

Regras para a v1:

- manter os arquivos escritos por humanos e versionados no repositório
- usar arquivos texto simples ou Markdown
- tratar isso como conteúdo-fonte, não como saída gerada
- reconstruir o índice RAG após alterar arquivos aqui
- preferir linguagem de negócio/processo em vez de recitação de schema
- incluir ressalvas realistas, notas de reporting e orientações de interpretação
- mencionar estruturas de dados apenas quando isso surgiria naturalmente em um documento voltado para humanos
